🌿 Sisäkasvien kastelukalenteri

Automaattisesti päivittyvä kalenteri sisäkasvien kastelu-, lannoitus- ja kukintamuistutuksille.

📥 Lataa kalenteri

Tilaa kalenteri (suositeltu)

Jotta kalenteri päivittyy automaattisesti, tilaa se kalenterisovellukseesi alla olevan URL:n avulla. Älä tuo tiedostoa, vaan käytä "Tilaa kalenteri" tai "Subscribe to calendar" -toimintoa.

https://1nri.github.io/sisakasvien-kastelukalenteri/sisakasvien-kastelukalenteri.ics

Ohjeet eri sovelluksille:

Google Calendar:

  1. Avaa Google Calendar
  2. Vasemmalla: "Muiden kalenterien" vieressä klikkaa +
  3. Valitse "URL-osoitteesta"
  4. Liitä yllä oleva URL ja klikkaa "Lisää kalenteri"

Apple Calendar (macOS/iOS):

  1. Avaa Kalenteri-sovellus
  2. Valitse Tiedosto → Uusi kalenterin tilaus (macOS) tai Asetukset → Tilit → Lisää tili → Muu → Lisää tilattu kalenteri (iOS)
  3. Liitä yllä oleva URL
  4. Valitse päivitystiheys (suositus: päivittäin tai viikoittain)

Microsoft Outlook:

  1. Avaa Outlook-kalenteri
  2. Klikkaa Lisää kalenteri → Tilaa internetistä
  3. Liitä yllä oleva URL

Sisältö

Kalenteri sisältää muistutuksia seuraaville kasveille:

  • Rahapuu (Crassula ovata)
  • Sutipuu (Pachira aquatica)
  • Kiiltojukka (Yucca gigantea)
  • Rönsylilja (Chlorophytum comosum)
  • Kirjovehka (Dieffenbachia seguine)
  • Kirjojuoru (Tradescantia zebrina)
  • Anopinkieli / Anopinhammas (Dracaena trifasciata)
  • Limoviikuna (Ficus benjamina)
  • Liuska-aralia (Schefflera arboricola)
  • Kultaköynnös (Epipremnum aureum)
  • Afrikkalainen maitopensas (Synadenium grantii)
  • Viirivehka (Spathiphyllum spp.)
  • Pallerokirjotähti (Haworthia attenuata)
  • Meksikonruusu (Echeveria elegans)
  • Hopeatolppakaktus (Cleistocactus strausii)
  • Samettianthurium (Anthurium crystallinum)
  • Saniainen (Nephrolepis exaltata)
  • Palmuvehka (Zamioculcas zamiifolia)
  • Ruusubegonia (Begonia x hiemalis)
  • Pelargoni (Pelargonium x hortorum)
  • Basilika (Ocimum basilicum)
